## Project Overview
FinPay is a modern fintech mobile app UI designed
with 5 core screens — Login, Dashboard, Send Money,
Card Management, and Analytics.

## Files in this Repository
| File | Description |
|------|-------------|
| FinPay_Report.pdf | Full design report (2–3 pages) |
| FinPay_Wireframe.png | B&W wireframe — 5 screens |
| FinPay_UI.png | Colour UI design — 5 screens |

## Design Highlights
- Dark navy theme with violet accent (#7C6FFF)
- KYC verified login with biometric support
- Real-time balance card with sparkline
- Card freeze, toggle controls, spending limits
- Weekly analytics with category breakdown
